Introduction

The All-In-One IPC and RPC code samples demonstrate the typical programming topics about interprocess communication (IPC) and remote procedure call (RPC). These topics are either frequently asked in MSDN forums and Microsoft Support, or are requested by many developers via our code sample request function. We want to alleviate the frustration felt by developers when they are using IPC and RPC technologies.
