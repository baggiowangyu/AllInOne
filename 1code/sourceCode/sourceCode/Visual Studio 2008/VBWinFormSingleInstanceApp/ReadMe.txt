/////////////////////////////////////////////////////////////////////////////
Use:

The SingleInstanceApp sample demonstrates how to achieve the goal that only 
one instance of the application is allowed in Windows Forms application. In 
this sample, we are taking advantage of VB.NET Framework since it has built
-in support for single instance application.It is quite difference form us-
ing in c#.

/////////////////////////////////////////////////////////////////////////////
Code Logic:

   1. Create a MainForm.
   2. Open the property window of the project.
   3. Select the Application tab, there is a groupbox named windows applica-
   tion framework property.
   4. In that groupbox, select the "Make single instance application" che-
   ckbox .
   5. Built the project.
